# EverythingAgent 使用教程

本教程详细介绍 EverythingAgent 的各项功能使用方法，帮助您充分发挥这款智能桌面助手的强大能力。

## 🚀 快速入门

### 基本操作流程

#### 1. 唤醒应用
- 按下全局快捷键 `Alt+Space`
- 应用窗口会在屏幕中央弹出
- 输入框自动获得焦点

#### 2. 基本交互
- **输入文本**：直接在输入框中输入内容
- **发送指令**：按 `Enter` 键确认
- **关闭窗口**：按 `Esc` 键隐藏窗口

#### 3. 窗口操作
- **移动窗口**：拖拽顶部的抓手区域
- **调整大小**：窗口会根据内容自动调整
- **系统托盘**：最小化到托盘，右键可访问菜单

## 💬 AI 对话功能

### 基本对话使用

#### 开始对话
1. 唤醒应用（Alt+Space）
2. 直接输入您的问题或指令
3. 按 Enter 发送

**示例对话**：
```
用户：今天天气怎么样？
AI：我需要获取您所在地区的天气信息...
```

#### 对话特点
- **流式输出**：AI 回复会逐字显示，如同真人打字
- **上下文保持**：支持多轮连续对话
- **智能理解**：能够理解上下文关联

### 高级对话技巧

#### 1. 明确指令格式
使用清晰的指令格式获得更好效果：

✅ 推荐格式：
```
请帮我解释一下 React Hooks 的原理
我想了解 Python 中装饰器的用法
帮我写一个计算斐波那契数列的函数
```

❌ 不推荐格式：
```
hook是什么
装饰器
fibonacci
```

#### 2. 上下文管理
- 单次对话建议控制在 5-10 轮以内
- 复杂话题可分多次对话
- 使用「新对话」按钮开始全新话题

#### 3. 专业领域对话
针对不同领域调整表达方式：

**编程相关**：
```
请用 JavaScript 实现一个防抖函数
解释一下数据库索引的工作原理
帮我优化这段 SQL 查询语句
```

**写作相关**：
```
帮我润色这篇技术博客文章
写一段产品介绍文案
总结这份会议纪要的要点
```

**学习相关**：
```
用通俗易懂的方式解释量子力学
列出学习机器学习的学习路径
推荐一些关于心理学的经典书籍
```

### 对话历史管理

#### 查看历史对话
1. 点击输入框右侧的 🕒 历史按钮
2. 浏览之前的对话记录
3. 点击任意对话可继续

#### 管理对话记录
- **删除对话**：悬停在对话项上，点击垃圾桶图标
- **新建对话**：点击 ➕ 新建按钮
- **搜索对话**：在历史面板中直接搜索关键词

## 🔍 文件搜索功能

### 基本搜索操作

#### 触发搜索模式
在输入框中使用以下任一前缀：
- `> 关键词` （推荐）
- `find:关键词`
- `search:关键词`

**示例**：
```
> report.docx
find:项目计划
search:2024年总结
```

#### 搜索结果显示
- 实时显示匹配的文件和文件夹
- 支持键盘导航选择
- 显示文件路径和类型图标

### 搜索操作技巧

#### 1. 精确搜索
```
> "exact filename.txt"    # 精确匹配文件名
> *.pdf                   # 搜索所有 PDF 文件
> folder/                 # 搜索特定文件夹
```

#### 2. 路径搜索
```
> C:\Users\Documents\*.docx    # 指定路径搜索
> project/src/*.js            # 项目源码搜索
```

#### 3. 组合搜索
```
> report 2024 AND .xlsx       # 同时包含多个关键词
> (urgent OR important) AND meeting  # 逻辑组合搜索
```

### 搜索结果操作

#### 打开文件
- **Enter**：用默认程序打开选中文件
- **鼠标左键单击**：同上

#### 定位文件
- **Ctrl+Enter**：在资源管理器中显示文件位置
- **鼠标右键单击**：同上

#### 快速导航
- **↑↓ 方向键**：在搜索结果间移动
- **PageUp/PageDown**：快速翻页
- **Home/End**：跳转到首尾

### 搜索高级功能

#### 文件类型过滤
```
> *.jpg                   # 仅图片文件
> *.pdf OR *.docx         # 文档文件
> NOT *.tmp               # 排除临时文件
```

#### 时间范围搜索
```
> modified:last week      # 最近一周修改的文件
> created:today           # 今天创建的文件
```

## ⌨️ 快捷键大全

### 全局快捷键
| 快捷键 | 功能 |
|--------|------|
| `Alt+Space` | 唤醒/隐藏应用（默认） |
| `Ctrl+Alt+Space` | 备用快捷键 |
| `Win+Shift+A` | 自定义快捷键 |

### 窗口内快捷键
| 快捷键 | 功能 |
|--------|------|
| `Esc` | 关闭搜索/收起对话/隐藏窗口 |
| `Enter` | 发送消息/打开选中文件 |
| `Ctrl+Enter` | 在资源管理器中定位文件 |
| `↑↓` | 在搜索结果中导航 |
| `Tab` | 在界面元素间切换焦点 |

### 对话相关快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建对话 |
| `Ctrl+H` | 显示/隐藏历史 |
| `Ctrl+S` | 保存当前对话 |
| `Ctrl+D` | 删除当前对话 |

## 🔧 故障排除与优化

### 常见问题解决

#### 搜索无结果
```
检查事项：
✓ Everything 服务是否运行
✓ 搜索关键词是否正确
✓ 文件是否在索引范围内
✓ 重启 Everything 服务
```

#### AI 回复异常
```
解决步骤：
1. 检查网络连接
2. 验证 API Key 有效性
3. 尝试切换其他模型
4. 查看开发者工具控制台错误信息
```

#### 性能优化建议
- 定期清理对话历史
- 关闭不常用的 AI 模型
- 调整搜索结果显示数量
- 重启应用释放内存

### 最佳实践

#### 高效使用技巧
1. **善用快捷键**：熟练掌握各种快捷键操作
2. **精确表达**：对话时尽量明确具体的需求
3. **合理分类**：不同类型的任务使用相应的模式
4. **定期整理**：及时清理不需要的对话和文件

#### 工作流程整合
